# Discord Encuestas Bot

Bot de Discord para gestionar encuestas con asistencia.

## Funciones
- Encuestas con botones
- Motivo obligatorio si llegas tarde
- Cambio de respuesta
- Recordatorio automático 30 min antes
- Solo administradores pueden usarlo

## Uso
/encuesta texto:"..." hora:"20:00" rol:@Rol
